# 🏠 House Price Prediction App

## 📌 Project Overview
This project is an end-to-end Machine Learning application that predicts house prices based on user inputs like area, year, number of rooms, and floor.

---

## 🚀 Features
- Predict house prices using Linear Regression
- Interactive web app using Streamlit
- User-friendly interface
- Real-time predictions

---

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

---

## 📊 Input Features
- Area (sq ft)
- Year
- Number of Rooms
- Floor
